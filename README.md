# skeletons.vim
> Simple snippet insert plugin for my own use.

Includes the following boilerplate snippets:
- VueSkeleton

## Install with vim-plug
```
Plug 'tristan-smith/skeletons.vim'
```
```
:PlugInstall
```

## Use it
```
:VueSkeleton
```

## Remap It
```
if g:skeletons
  nnoremap ,vue :VueSkeleton<CR>
endif
```
